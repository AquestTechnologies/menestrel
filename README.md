# Ménestrel
*Tell stories like it's 1390*

Ménestrel is a storytelling javascript library.

download and run example : 
```
mkdir menestrel
git clone https://github.com/AquestTechnologies/menestrel.git ./menestrel
npm i
cd ./example && npm i && cd ..
npm run example
```
